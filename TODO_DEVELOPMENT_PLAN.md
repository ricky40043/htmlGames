# 🎮 靜態桌遊平台開發計劃

## 📋 專案概述
開發三個新的桌遊：
1. **你問我答** - 2人問答對戰遊戲 (500題庫)
2. **2種人** - 社交分類性格測試遊戲 (200題庫)  
3. **誰是臥底** - 多人QR Code推理遊戲

---

## 🚀 建議開發順序
1. ✅ 建立開發計劃文件
2. 🔄 2種人遊戲（最簡單，純選擇邏輯）- **進行中**
3. ⬜ 你問我答（中等複雜度，需題庫管理）
4. ⬜ 誰是臥底（最複雜，需QR Code整合）
5. ⬜ 首頁整合（最後統一整合）
